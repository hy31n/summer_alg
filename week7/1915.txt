1915번 가장 큰 정사각형(골드)
1. 문제 인식
N x M의 0, 1로 된 배열이 있다. 이 배열에서 1로 된 가장 큰 정사각형의 크기를 구하는 프로그램을 작성하시오.

2. 문제 유형
사각형의 기준점에서 왼쪽, 오른쪽, 왼쪽 대각선의 최솟값이 1이면
기준점의 수를 2로 변경한다.

3. 의사 코드
N x M을 입력받아서 dp 배열을 입력받음
N : 배열의 세로, M: 배열의 가로, max : 최댓값 저장

dp[i][j](현재 자리)가 0이 아닌 점화식
- dp[i][j] = MIN(dp[i - 1][j - 1], dp[i][j - 1], dp[i - 1][j] + 1) (현재 자리의 원래 값 + 1) 

dp[i][j]의 값이 1일 때, 자신의 위, 왼쪽, 왼쪽 대각선 + 1을 저장
만약 dp[i][j]가 최댓값보다 크면 최댓값 갱신

dp[i - 1][j - 1] : 대각선
dp[i - 1][j] : 왼쪽
dp[i][j - 1] : 위
