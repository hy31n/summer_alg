1068번 트리(골드)
문제 인식 : 트리가 주어졌을 떄. 노드 하나를 지울 것이다.
            그때, 남은 트리에서 리프 노드의 개수르 구해라.

문제 유형 : 트리에서 노드를 지우면 지운 노드의 자식들도 모두 지워진다.
            부모가 없는 노드를 루트 노드라고 하고 루트 노드부터 DFS를 진행한다.
            그 이외에는 제거한 노드를 기억하여 DFS를 실행할 때 방문하지 않는다.
            리프 노드는 자식이 없는 노드이므로 vector 사이즈를 확인하여 0이면 count++한다.

의사 코드 : if (0 == tree[cur].size()) 자식이 없으면 리프 노드
            if (1 == tree[cur].size() && nodeToRemove == tree[cur][0]) 자식이 하나인데 지워야 할 노드라면 현재 노드가 리프 노드
            main
            - 트리의 크기 N을 입력받고 각 노드의 부모를 입력받는다.
              입력이 -1이라면 루트 노드다.
              지울 노드를 입력받는다.
