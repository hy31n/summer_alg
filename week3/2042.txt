2042번 구간 합 구하기
문제 인식 : 여러 N개의 수가 주어져 있고, 수가 빈번하게 수정될 때,
            구간 합을 효율적으로 구해야 하는 문제이다.

문제 유형 파악 : dp 배열(0에서 i번 index까지 합을 저장해두는 배열)을 사용하면 배열의 값이 자주 바뀐다면
                구간 합을 갱신하기 위해 값을 바꿀 때 마다 O(N)의 연산이 필요하고
                값을 바꾸는 쿼리가 n번 있으면 O(N^2)의 시간복잡도를 가진다.
                이는 오버헤드가 크기 때문에 세그먼트 트리를 써야한다.

의사 코드 : 수의 개수 N, 수의 변경이 일어나는 횟수 M, 구간 합을 구하는 횟수 K 입력 받음
            세그먼트 트리를 구현하기 위해 값을 업데이트 해주는 update함수와
            query함수를 정의한다.
