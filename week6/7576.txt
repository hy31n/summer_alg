7576번 토마토(골드)
1. 문제 인식
격자모양 상자의 칸에 토마토를 하나씩 넣어 보관한다.
익은 토마토(1)와 인접한 곳에 있는 익지 않은 토마토(0)들은 익은 토마토(1)의 영향을 받아 익는다.
상하좌우 네 방향에 있는 토마토가 영향을 받는다.
익지 않거나 익은 토마토가 하나도 없는 경우도 있다.
모든 토마토가 익을 때까지의 최소 날짜를 출력하자.

2. 문제 유형
인접한 곳 -> BFS 
상하좌우에 대해 접근하는 함수를 생각함

3. 의사 코드
int dx[4] = { 1,-1,0,0 }
int dy[4] = { 0,0,1,-1 }
-> 상하좌우 설정
