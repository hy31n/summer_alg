2193번 이친수(실버)
1. 문제 인식
0과 1로만 이루어진 수를 이진수라 한다. 이친수는 다음의 성질을 만족한다.

이친수는 0으로 시작하지 않는다.
이친수에서는 1이 두 번 연속으로 나타나지 않는다. 즉, 11을 부분 문자열로 갖지 않는다.

2. 문제 유형
N = 1 : { 1 }
N = 2 : { 10 }
N = 3 : { 100, 101 }
N = 4 : { 1000, 1001, 1010 }
N = 5 : { 10000, 10001, 10010, 10100, 10101 }
D[N] = D[N - 1] + D[N - 2] 규칙이 반복된다.
